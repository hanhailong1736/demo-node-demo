# node-youxi
## 游戏的后台项目

> 作者：韩海龙 <hlmyprivate@163.com>

### 环境依赖

node v8.4.0+


### 部署步骤

1. npm install        //安装node运行环境
2. npm run dev        //开发环境构建
3. npm run start      //开发环境构建 热更新
4. npm run build      //生产环境构建
